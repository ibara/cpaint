cpaint
======
A console painting program.

Building
--------
Just run `make`. You'll need the `ncurses` library.

Controls
--------
hjkl: move cursor
c: change color
s: save
q: quit

This isn't C code!
------------------
That's a matter of interpretation, really.
See [this post](https://briancallahan.net/blog/20220220.html) for more.
